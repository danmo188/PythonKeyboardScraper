Features:
	This simple application will count how many times you press an alpha-numeric key
	and log it into a file. 

	Runs in the background so it doesn't cause a hassle

Basic setup:
	You can start the program every time you start up your computer
	but it's much easier to add it to automatically start 

	HOW TO ADD PROGRAMS TO START AUTOMATICALLY WINDOWS 10:
		create a shortcut of the executable or script

		in our case make a shortcut of start.vbs, located in the dist folder
		place the short in

		C:\Users\<username>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

		now the program will run automatically.
